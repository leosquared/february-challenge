# february-challenge

### Tech Specs
- Scripts and Jupyter Notebooks are executed in *Python 3.6*
- Python dependencies can be found in [requirements.txt](https://github.com/leosquared/february-challenge/blob/master/requirements.txt)
- Local database is *PostgreSQL 10.2*, and local credentials are saved in a YAML file, omitted in this repo

### Scripts
- [Data Exploration Notebook](https://github.com/leosquared/february-challenge/blob/master/data-files-exploration.ipynb) is a short exploration to get a glimpse of the files
- [psql_etl.py](https://github.com/leosquared/february-challenge/blob/master/psql_etl.py) is a script to perform all of the ETL tasks specified in the challenge, including filling in blank columns
- [Analysis & Visualization Notebook](https://github.com/leosquared/february-challenge/blob/master/analysis-visualization.ipynb) is a series of scripts aimed at gaining data insights around customer purchasing behavior and the loyalty program, incuding the visualizations used in the presentation
